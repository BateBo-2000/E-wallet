==============================
How to run E-wallet project
==============================

Prerequisites:
--------------
1. Node.js installed on your system.

Instructions:
--------------

1. Open a terminal/command prompt.

2. Navigate to the project's root directory.
   (The directory containing the 'client' and 'server' folders)

3. Run the server:

   a. Change directory to the server folder:
      > cd server

   b. Install server dependencies:
      > npm install

   c. Start the server:
      > npm start

   d. Leave the terminal/command prompt running and open a new terminal/command prompt.

4. Run the client:

   a. Change directory to the client folder:
      > cd client

   b. Install client dependencies:
      > npm install

   c. Start the client:
      > npm run start

5. The E-wallet project should now be running with the server and client applications.
